# Logo MyTugas

Silakan tambahkan 2 file logo berikut ke folder ini:

1. **mytugas_logo.png** - Logo dengan background (untuk icon aplikasi)
2. **mytugas_logo_transparent.png** - Logo tanpa background/transparent (untuk login screen)

Ukuran yang disarankan:
- mytugas_logo.png: 1024x1024 px (untuk icon aplikasi)
- mytugas_logo_transparent.png: Minimal 512x512 px (untuk login screen, akan di-resize otomatis)

Pastikan file-file tersebut berada di folder `assets/images/` setelah ditambahkan.
